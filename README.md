## 🧩 Les 14 Exercices Pratiques

---

### ✅ Exercice 1 : Structure HTML de base

**Niveau : Débutant | Temps : 15 min**

Créer une page HTML5 valide contenant :

* Titre : *"Ma première page"*
* Un paragraphe de présentation
* Une liste de 3 hobbies
* Un lien vers Google (nouvel onglet)

**Objectif pédagogique :** comprendre la structure HTML5 minimale.

---

### ✅ Exercice 2 : Carte de visite

**Niveau : Débutant | Temps : 20 min**

Créer une carte HTML avec :

* Photo placeholder
* Nom et prénom
* Profession
* Email cliquable
* Téléphone cliquable
* Adresse avec lien Google Maps

**Objectif pédagogique :** maîtriser les liens spéciaux (`mailto`, `tel`).

---

### ✅ Exercice 3 : Menu de navigation

**Niveau : Intermédiaire | Temps : 25 min**

Créer un menu responsive :

* Logo à gauche
* 4 liens centrés
* Bouton Contact à droite
* Hover effects modernes
* Mobile : menu vertical

**Objectif :** Flexbox + responsive.

---

### ✅ Exercice 4 : Formulaire d'inscription

**Niveau : Intermédiaire | Temps : 30 min**

Formulaire complet avec :

* Nom / Prénom côte à côte
* Email validé
* Mot de passe + indicateur force
* Date de naissance
* Genre
* Pays
* Checkbox conditions
* Validation HTML5 + CSS

**Objectif :** UX et validation native.

---

### ✅ Exercice 5 : Galerie Photos Grid

**Niveau : Intermédiaire | Temps : 25 min**

* Grid CSS
* 3 colonnes desktop / 2 tablette / 1 mobile
* Hover zoom + overlay
* Lazy loading
* Légende au survol

**Objectif :** CSS Grid avancé.

# 🚀 MINI-PROJET FINAL

## 🎯 Site Vitrine – Portfolio Professionnel

Créer un site portfolio complet, moderne et responsive pour un développeur freelance.

---

## 📐 Structure (Single Page – 5 Sections)

1. Header fixe + navigation smooth scroll
2. Hero section + Call To Action
3. À propos + barres de compétences
4. Portfolio filtrable (grid projets)
5. Formulaire de contact
6. Footer avec réseaux sociaux

---

## ⚙️ Fonctionnalités obligatoires

* Design mobile-first
* Menu hamburger mobile
* Dark mode
* Animations scroll (fade-in)
* Filtres portfolio
* Validation formulaire
* Bouton retour en haut
