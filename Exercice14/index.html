Mon Site Web

Bienvenue

Ce site supporte le mode sombre ! Cliquez sur l'interrupteur en haut à droite pour basculer entre les thèmes clair et sombre.

Fonctionnalités

Le toggle est entièrement géré en CSS grâce à la technique checkbox + label + sélecteur :checked. Aucune ligne de JavaScript n'est nécessaire.

Accessibilité

Si vous n'avez pas encore basculé le toggle, le site respectera les préférences de votre système d'exploitation via @media (prefers-color-scheme: dark).
